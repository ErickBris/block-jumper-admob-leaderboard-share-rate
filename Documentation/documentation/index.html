Game Speed

Changing Game Speed

The game settings allow you to change the game speed and speed increments.

Note; Line numbers may vary slightly depending on the version of the game.

Controlled via strings.xml, find this section;

 

 

Game Speed

Simply increase the speed for a harder game, decrease for any easier game.

 

 

Game Speed Increments

This controls the frequency of the speed increase. The default is set at 20. This means ever 20 points the game speed will increase.

Set this value to anything you want. High numbers mean an easier game, low numbers means the game speed will get faster with low scores.

 

 

Game Speed Increment Increase

This sets the percentage speed increase per increment. The default is set to 5, that means every increment will increase game speed by 5%.

Higher numbers mean a harder game, lower numbers mean an easier game.

Share Text

Changing Share Text

All handled via strings.xml, find this section;

 

There are three variables you can use:

  1. @score@ displays the players highest score.
  2. @appName@ displays the game's name.
  3. @gameUrl@ displays the game's url as found in the play store.

Google Settings

Changing the Google Settings

Note; Line numbers may vary slightly depending on the version of the game.

All handled via strings.xml, find this section;

 

 

Setting up Google Play for Your App

Google have already created a comprehensive tutorial on the subject, it can be found here: 

https://developers.google.com/games/services/console/enabling

 

 

Setting the app_id

Simply add your app_id as supplied by Google to app_id

 

 

Setting Leaderboard

Add your leaderboard id supplied by Google to leaderboard_leaders

 

 

Admob Id

admob_id is your admob is supplied by Google

 

 

Interstitial Ad Id

InterstitialAd_unit_id is the ad id of your interstitial ad as supplied by Google

 

 

Interstitial Ad Frequency

InterstitialAd_frequency sets the frequency of interstitial ad display. A value of 3 means 1 in every 3 deaths will show these ads. A value of 1, means every death will show these ads.

 

 

 

Banner Placement

Toggle banner place on top or bottom, you can not have both and the system will default to bottom in the event that both values are set to true.

 

 

Admob Interstitial On

Toggle interstitial ads on or off.
